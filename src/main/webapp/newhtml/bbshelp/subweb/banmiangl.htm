版面的管理条例
第一、申请开设新版面：  
一、本站合法ID，可依本规定在“问题意见”版提出开版申请。
二、申请文章要求：
  1、标题。 请用 "[申请]开设新版" 作为标题。
2、内容.。内容请包括：
(1) 、申请版的中文名
中文标题不得超过 4个中文字。
(2) 、版面分类
说明希望该版属于哪一分类，即放在哪一个区。
(3)、 版面说明
即版面宗旨, 具体的说明开设新版的目的、以及讨论的主要内容和要
发展的方向。
(4)、 提议第一任版主, 该版主要符合本站“版主申请”的规定, 其本人要按本
站申请版主规定另外提交版主申请书。
(5) 、提议人署名 。
第二、关于开版的规定  
一、基本原则:
  1. 不开设违反国家法律, 学校规章制度, 本论坛站管理办法的版块。
2. 不开设有违反社会公德与良好风俗的版块。
3. 不重复开设性质相近或目的相同的版块。 如果新版块的主题含混不清，或者
与已有的版块界限不明，可能造成网友发表文章不能确定对应的版块时，
不予开设。
4. 内容过于生僻、狭窄，讨论人数不会很多的版块不考虑开设。
5. 主题对本站不利者, 不考虑开设。
6、参照网友的意见。
二、连署原则
  1. 必须回复原文，另起标题回复者无效；
2. 回复文须加上自己ID。
3. 回复必须明显表示支持或反对, 无论反对与赞同都请说明原因。
4. 同一人用多个账号回复则其所有回复均无效。
5. 被提议的版主必须参加连署，并表示支持。
6. 一周内20人以上有效支持且不超过8人有效反对则提交站务组讨论。
三、网管部一票否决原则
  网管部有权驳回申请人申请, 并说明理由。
四、特殊原则
  若有其他性质特殊原因开辟特殊的版块，由站务会议决定处理方式。 经站务管理组讨论同意,可由站务公告说明后直接开设。
第三、关闭版块  
一、 版块关闭成立的条件：
  1. 版块的性质或讨论的内容违反雨无声网站管理办法,或者版块已悖离当初开版宗旨者。
2.版块使用率 (非文章数量) 过低者。
3. 版块在实际内容上有重复性质，可考虑合并与关闭。
4. 因其他不可抗力因素必须关闭的版块，由网管部决定。
二、凡符合关闭条件，经网管部决议后， 通知该版版主，并公告关版限期，到期关版。
第四、版块名称变更  
一、版块名称的变更，例如版块的中文版名， 可由版主于该版提出讨论。 有结果后，版主可以将更改的理由及讨论结果于“问题意见”版向网管部提出申请。申请需以“[申请] 变更[原版名]为[新版名]”为题发表于“问题意见”版，内容格式如后：
  1. [版块名称]：
2. [版主]：
3. [申请事项]：
4. [说明及讨论结果]：
二、提出申请的变更，需符合开版条件，且不得构成关版条件。
三、申请的版面是否变更，由网管部会议决定。自正式提出申请之日起， 变更所需时间为一个星期。如遇特殊情况，变更作业时间可延长，具体情况由站务人员在原贴中跟贴说明。
四、版的属性经网管部商议后，可通知该版版主，予以变更。
第五、版块的津贴  
每一个版面将有2W西大币作为奖金，奖励给这一个版块表现好的网友，由这一个版块的版主在“版主之家”提出申请。
申请格式为：
  1）申请人:
2)申请资金金额：
3）资金用途。
如遇特殊情况，给予版块的版块津贴性质将不一样。
第六、其他  
一、申请失败和遭否决之申请提案，一个月内不得提出申请成立相同性质的版块。
二、对于讨论秩序出现严重问题的版面 (比如恶性灌水、争吵),网管部有权临时作出立即设置该版为“权限”状态的决定，并且进行有效的整理。这样的临时设置“权限”，为期最长不能超过三天。
第七、本管理办法解释权属于雨无声网管部所有，并有权视情随时更改。有关事项 与其他管理办法互为补充。自公布之日起执行
