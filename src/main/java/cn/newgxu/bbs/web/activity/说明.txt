activity 包是论坛中的活动应用。

比如很快就要来到的 11.11 光棍节，就是一个应用。 应用只是存在于某几天，有点会有自己的发帖页面。
应用的 action，model，service 都是继承于论坛中原来的类。

添加应用时，可以直接继承  ActivityAction.java 类，一个应用对应一个action类，尽量不要产生多个action类
活动中需要添加Item的，请要Item 中备注是哪个活动的。

由于考虑到某些活动可能会对用户的名称添加额外信息（比如光棍节在结婚的两个用户中，使用了结婚证道具后，在名称后有一个结婚的小图标，title 为XXX的丈夫/夫人）
会有一个管理活动称谓的类：Activity.java

Activity.java 是管理 活动的总体类，可以获取一个活动的开始/结束时间
